
## LAKRA
LLM_AUG codebase provides the implementation to reproduce the LLM-AUG module.
```text
LLM_AUG/
├─ tail.py    #Find the entity with the lowest degree
├─ rel_entity_Type_dict.py.txt    #For candidate set generation
├─ long_tail_counter.py    #Count the statistics of long-tail entities
├─ extract_khop_subgraph.py    #Extract k-hop subgraph for low-frequency entities
├─ calculate_shortest_path.py    #statistic shortest distance between entities
|
├─ llm/
│  ├─ main.py    #LLM-AUG module entry point
│  └─ utils_.py     #LLM-AUG module utility functions
│  
├─ sparse_nodes/   #Stores k-hop subgraphs for low-frequency entities
|  |...
|  ├─ WN18RR/
│  └─ fb15k-237/
│ 
├─ generated_triplets/  # Final consolidated triplets generated by the large model
│  |...
```


Please run the following command to reproduce the reported results on FB15k-237 dataset

```python
CUDA_VISIBLE_DEVICES=0 python main.py --train_path /path/to/traindata/folder/ --test_path /path/to/testdata/folder/ --ent_num 14541 --rel_num 474 --cl_epochs 1500
```
